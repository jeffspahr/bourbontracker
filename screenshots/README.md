# Screenshots

This directory contains screenshots for the README.

## Required Screenshots

Please take the following screenshots and save them in this directory:

### 1. map-overview.png
- **What to capture:** Full map view showing multiple store markers across Virginia
- **How to take it:**
  1. Open http://localhost:8000/map.html
  2. Ensure the map shows all products (no filter applied)
  3. Zoom out to show a good portion of Virginia with multiple markers visible
  4. Take a screenshot showing the header, stats bar, filter bar, and map
- **Recommended size:** 1600x1200 or larger

### 2. product-filter.png
- **What to capture:** The product filter dropdown expanded showing available products
- **How to take it:**
  1. Click on the "Filter by Product" dropdown to open it
  2. Ensure several products are visible in the list
  3. Take a screenshot showing the dropdown menu with product counts
- **Recommended size:** 1600x1200 or larger
- **Note:** Should show the format "Product Name (X bottles at Y locations)"

### 3. store-detail.png
- **What to capture:** A store marker's info window showing product details
- **How to take it:**
  1. Click on a store marker (preferably one with multiple products)
  2. Wait for the info window to appear
  3. Take a screenshot showing the info window with product details and quantities
- **Recommended size:** 800x600 or larger
- **Note:** Choose a store with interesting products like Pappy Van Winkle or Blanton's if available

## Screenshot Tips

- Use high resolution (at least 1440p)
- Ensure good contrast and readability
- Show the full interface (don't crop too tightly)
- Use PNG format for best quality
- Compress images if they're over 500KB (use tinypng.com or similar)

## Quick Command to Take Screenshots

### macOS
```bash
# Full screen
Cmd + Shift + 3

# Selected area
Cmd + Shift + 4

# Window capture
Cmd + Shift + 4, then press Space, then click window
```

### Linux
```bash
# Using gnome-screenshot
gnome-screenshot -w  # Window
gnome-screenshot -a  # Area

# Using scrot
scrot -s  # Select area
```

### Windows
```bash
# Snipping Tool or Snip & Sketch
Win + Shift + S
```
